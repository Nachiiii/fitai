# FitAI — AI Fitness Coach

Your personal AI-powered fitness app. Users set their goals, upload a photo, and get a custom workout + diet plan.

## Features
- Bulk or Cut goal setting
- Home 🏠 or Gym 🏋️ workout plans (switchable anytime)
- AI-generated 7-day workout plan
- AI-generated daily diet plan
- Personalized tips
- Optional body photo analysis

---

## Deploy to Vercel (Free — takes 5 minutes)

### Step 1 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 2 — Upload your files
1. Click the **+** button → "New repository"
2. Name it `fitai`
3. Click "uploading an existing file"
4. Drag and drop `index.html` and `vercel.json`
5. Click "Commit changes"

### Step 3 — Deploy on Vercel
1. Go to https://vercel.com and sign up with your GitHub account
2. Click "Add New Project"
3. Select your `fitai` repository
4. Click **Deploy** — no settings needed!
5. Vercel gives you a live URL like `fitai.vercel.app` 🎉

## Your app is now live!
Share the link with anyone. It works on phones, tablets, and computers.

## To update the app later
Just edit `index.html` on GitHub and Vercel will automatically redeploy.
